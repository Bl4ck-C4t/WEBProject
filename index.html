Basics

Using Control flow 

a = 10
for(x in range(a)){
  if(x % 2 == 0){
    print(x)
  }
  else if(x == 10) {
    print("10 reached")
  }
  else{
    print("something else")
  }
}

while(True){
  print("Greetings humans")
}

Using and and or Allows for multiple conditions in if

The default console output function is print(string text)
The default input function is input()

Iteration and iterables

There is only 1 type of iterable currently - The dynamic list 

List ls = []
ls.push(10)
ls.push(20)
for (c in ls){
  print(c)
}

Lists can be filled with any type or constrained to a specific type:List<int> ls
This also allows iteration by type for(int a in ls)
The default list allows any type, however variable assignment can still happen even on mixed type lists. 

List ls = []
ls.push(15)
ls.push(20)
for (int c in ls){
  print(c)
}
